# 🚀 BuildWise AI

BuildWise AI is an AI-powered software project mentor that helps developers and students generate professional project ideas instantly.

## ✨ Features

- 🤖 AI Project Idea Generator
- 📚 AI Development Roadmap
- 🛠️ Recommended Tech Stack
- 💼 Recruiter Feedback
- 📄 AI Generated GitHub README
- ⚡ Modern Responsive UI
- 🎥 Demo Video Support

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Framer Motion
- Axios
- React Markdown

### Backend
- Node.js
- Express.js
- OpenRouter AI API
- Axios

## 🚀 Installation

### Clone

```bash
git clone https://github.com/kavinilaM02/buildwise-ai.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm start
```
